## Solution
제가 생각했을 때 원래 문제의 의도는 스크립트의 동적 분석인 것 같습니다. 아래와 같은 코드를 작성하여 스크립트를 돌려주고 크롬 개발자도구 console 창을 열어보면 플래그가 찍혀있습니다.

```
<!DOCTYPE html>
<html>
<head>
	<title> HackCon Not Web Solution</title>
</head>
<body>
<script src="./ihatejs.js">
</script>
</body>
</html>
```
